Summary
---------------
Companion Android application to the Sight Cycle bike light. Named AT&T Best
M2X Application and AT&T Community Choice app.

Features
---------------
 - Fall detection using accelerometers
 - [AT&T M2X](https://m2x.att.com) tie-in

Platforms
---------------
 - Android (min API 15)
