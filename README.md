Summary
---------------
Companion Android application to the Sight Cycle bike light. Awarded Best Use of
the M2X API.

Features
---------------
 - Fall detection using accelerometers
 - [AT&T M2X](https://m2x.att.com) tie-in

Platforms
---------------
 - Android (min API 15)
